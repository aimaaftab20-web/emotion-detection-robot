# Emotion Detection Robot using Raspberry Pi

This project presents an intelligent emotion detection system that identifies human facial emotions in real-time using computer vision and machine learning.

## 🚀 Features
- Real-time face detection using OpenCV
- Emotion classification using TensorFlow/Keras
- Live video processing with Raspberry Pi camera module
- Displays detected emotions for interactive human–machine response

## 🛠 Technologies Used
- Python
- OpenCV
- TensorFlow
- Raspberry Pi 4
- Computer Vision

## 🎯 Applications
- Human–Robot Interaction
- Smart Surveillance Systems
- Healthcare Monitoring
- Educational Robotics
